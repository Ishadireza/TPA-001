Mengenal CSS

4 April 2023 / Tech

CSS (Cascading Style Sheets) adalah salah satu teknologi inti dari web design dan pengembangan web. CSS digunakan untuk mengatur tampilan dan layout halaman web, seperti menentukan warna, font, margin, padding, dan ukuran teks. CSS memisahkan tampilan dari struktur atau konten dari sebuah halaman web. Dengan menggunakan CSS, kita dapat memisahkan gaya (styling) dari HTML (Hypertext Markup Language), yang memungkinkan pengembang web untuk membuat tampilan yang sama untuk beberapa halaman web dengan mengubah kode CSS saja.

Mengenal HTML

4 April 2023 / Tech

HTML (Hypertext Markup Language) adalah bahasa markup yang digunakan untuk membuat dan mengatur konten pada halaman web. HTML memungkinkan pengguna untuk menentukan bagaimana konten pada halaman web harus ditampilkan, termasuk teks, gambar, video, dan audio. HTML berisi kumpulan elemen atau "tag" yang dapat digunakan untuk memformat dan menentukan tampilan konten pada halaman web. Setiap elemen HTML terdiri dari tag pembuka dan tag penutup, dan konten di antara tag tersebut.
